1.用inline-block布局
box1
box2
box3
      
        box是box2和box3的父容器，层级如下：
        box1
        box 
          box2
          box3
         
        .box {
            /* box2和box3之间默认会有换行符，即出现间隙，加了这句后间隙大小被设置为0，即去掉。但box中的所有字都不再显示 */
            font-size: 0;
        }

        .box1 {
            height: 100px;
            background-color: greenyellow;
        }

        .box2 {
            width: 50%;
            background-color: aquamarine;
            /* box2也要设为行内块状元素，否则box2还是要占一整行 */
            display: inline-block;
            /* 行内块级元素也要设置宽高，否则按默认值0处理 */
            height: 100px;
        }

        .box3 {
            width: 50%;
            background-color: rgb(130, 25, 179);
            display: inline-block;
            height: 100px;
        }
2.用float布局
box4
box5
box6
        
    .box4 {
        height: 100px;
        background-color: greenyellow;
    }

    .box5 {
        width: 50%;
        background-color: aquamarine;
        /* float脱离文档流，不占用空间，浮动在左侧,但可能遮盖其他元素，解决办法是在被遮盖的元素上加clear:both */
        float: left;
        height: 100px;
    }

    .box6 {
        width: 50%;
        background-color: rgb(130, 25, 179);
        /* 同样的float在左侧，紧贴在上一个float:left元素后面*/
        float: left;
        height: 100px;
    }
