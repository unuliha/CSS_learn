1.相邻边框连接处的均分原理，也就是相邻的边框相交的矩形各自占领一个三角形区域 
        
        #demo {
            width: 0;
            height: 0;
            border-width: 20px;
            border-style: solid;
            border-color: transparent transparent red transparent;
            }
2.更清楚的示例，这里 background-clip: content-box让背景颜色只作用于content 
        
        #demo1 {
            width: 10px;
            height: 10px;
            background-color: rgb(171, 185, 70);
            border-width: 20px;
            border-style: solid;
            border-color: red transparent red transparent;
            background-clip: content-box;
            }
